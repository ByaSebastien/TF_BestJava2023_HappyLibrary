Command

Créer un projet : ng new 'nom du projet'

Créer un module : ng g m 'nom du module'

Créer un component : ng g c 'nom du compenent'

Créer une interface : ng g i 'nom de l'interface'

Créer un service : ng g s 'nom du service'

Créer un pipe : ng g p 'nom du pipe'
